This is the two application for single-spa-angular cross domain test.

steps to start up:

1. cd app/app1
2. npm install
3. ng build
4. cd ../navbar
5. ng build
6. start up app on port 4200: npm start 
7. cd app2/app2
8. npm install 
9. ng build
10. start up app2 in port 3200: ng server --port 3200
